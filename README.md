# Snake-Game

Jeu Snake Vanilla JS multi-plateforme (PC/mobile/fullscreen).

## Fonctionnalités
- 4 modes : Facile/Moyen/Difficile/Crescendo (vitesse croissante)
- Contrôles clavier + swipe tactile
- Highscore localStorage
- Fullscreen + sons
- Loading screen

## Techno
Vanilla JS, CSS Grid, Fullscreen API, Touch events, Audio

## Modes
- **Facile** : 300ms
- **Moyen** : 200ms  
- **Difficile** : 100ms
- **Crescendo** : vitesse accélère jusqu'à "InSane"

## Démo
Ouvrir `index.html`

## Contrôles
- PC : Flèches
- Mobile : Swipe
